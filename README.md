# Multi-Level Water Level Indicator and Overflow Detection System

## Overview

Designed and developed a Multi-Level Water Level Indicator and Overflow Detection System using BC547 transistors, LEDs, resistors, water probes, and a battery-powered circuit. The system monitors water levels at different stages and provides visual indications for Low, Medium, High, and Overflow levels.

## Components Used

* BC547 Transistors
* LEDs (Low, Medium, High, Overflow Indicators)
* Resistors
* Water Probes
* 9V Battery
* Breadboard
* Connecting Wires

## Features

* Real-time water level monitoring
* Low, Medium, High level indication
* Overflow detection and alert
* Low-cost and energy-efficient design
* Easy implementation using discrete electronic components

## Applications

* Domestic water tanks
* Storage tanks
* Water management systems

## Skills

Analog Electronics, Circuit Design, Transistor Switching Circuits, Hardware Prototyping, Breadboard Design, Troubleshooting
